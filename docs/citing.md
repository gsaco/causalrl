# Citing CausalRL

If you use CausalRL in academic work, please cite the project. The repository
includes a `CITATION.cff` file with the latest metadata.

## How to cite

- Use the GitHub “Cite this repository” button.
- Or reference `CITATION.cff` directly in your citation manager.

## Plain-text citation

```
Saco, G. (2026). causalrl (Version 0.2.0). https://github.com/gsaco/causalrl
```

If you need a BibTeX entry, GitHub will generate one from `CITATION.cff`.
