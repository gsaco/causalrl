# Security Policy

## Supported Versions

We provide security fixes for the latest minor release line only.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a Vulnerability

Please email security reports to gsacow@gmail.com with:

- A clear description of the issue and impact.
- Reproduction steps or proof-of-concept code.
- Any suggested mitigations if you have them.

We will acknowledge receipt within 72 hours and work on a fix. Please do not
disclose vulnerabilities publicly until a fix is available.
